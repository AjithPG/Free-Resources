<h2>Gulp Install</h2>
<p>We need Node.js to run Gulp
https://nodejs.org/en/</p>


<p>Install gulp through cmd and Check gulp version</p>
<ul>
  <li>npm install gulp -g</li>
  
  <li>gulp --version</li>
</ul>

<p>Creating a task in gulp</p>
<code>
  gulp.task('task-name', function() {
   //do stuff here
});
</code>
